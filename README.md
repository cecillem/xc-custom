xc-custom
=========

1. Install Xcode
2. `ssh-keygen`
3. Install Homebrew
    ruby -e “$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)”
5. Copy Brewfile and brew bundle
    https://gist.github.com/cecillem/1630edea9227f9519126
7. Configure and install osxc.github.io
8. Install YADR
